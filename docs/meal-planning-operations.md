# Meal Planning Operating Model

## Weekly Workflow

1. Select weekly meals (menu)
2. Generate bulk prep tasks
3. Consolidate ingredients
4. Generate grocery list
5. Generate daily assembly guidance

All downstream outputs cascade from the finalized weekly menu.

---

## Evening Assistant

- Works Monday–Friday, 7–10pm
- Handles meal prep alongside other household duties
- Prep tasks must be time-efficient and clearly scoped

---

## Prep Scheduling Principles

Prep tasks are dynamically scheduled based on the week's menu — not assigned to fixed days.

**Core principles:**

- **Group heavy tasks together.** Identify which prep tasks can be batched (proteins, bases, sauces, veg) and schedule them on the same evening to minimize context switching and dishes.
- **Sequence prep to feed assembly.** Tasks should be scheduled so prepped ingredients are ready before they are needed — not the same night they're used.
- **Do incremental tasks as needed.** Smaller tasks (portioning lunches, trimming veg, pulling leftovers) are distributed across the week close to when they're needed.
- **Estimate active time.** Every prep night should include an estimated active time so the assistant can plan around other duties.
- **Instant Pot over slow cooker.** Proteins are cooked in the evening window — slow cooker timing doesn't fit.
- **Separate prep from assembly.** Prep is done by the assistant in advance; assembly happens at mealtime by the family.

### Example Scheduling Logic

Given a menu where shredded chicken is used Tuesday through Friday:
- Batch cook chicken on Monday evening (sets up the rest of the week)
- Pre-chop vegetables the same night while the protein cooks (efficient use of active time)
- Portion lunches each evening for the next day (incremental, ~10 min)

Given a menu where proteins vary each night with no shared base:
- No single heavy prep night needed
- Distribute smaller tasks across the week
- Focus evenings on portioning lunches and light ingredient prep

---

## Output Format for Prep Schedules

Each weekly prep plan should include:

- Which evening tasks are grouped on and why
- Estimated active time per night
- Clear separation between prep tasks and assembly tasks
- Notes on what the assistant can do versus what happens at mealtime
